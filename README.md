# Ray Portfolio MVP (Next.js)

## Run locally
```bash
npm install
npm run dev
```

Open http://localhost:3000

## Add your images
Put your photos here:
`public/images/`

Use these filenames (already wired up in the UI):
- `modular-car.jpg`
- `dyno.jpg`
- `robotic-arm.jpg`

## Deploy
- Vercel: import the repo and deploy
- Or self-host: `npm run build` then `npm run start`
